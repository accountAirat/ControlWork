## Поговорим о программе

### Первая функция ElementCounting
Задача этой функции посчитать количество элементов подходящих по наше правило
### Вторая функция FillNewArray
Задача этой функции записать элементы подходящие под наше правило в новый массив
### Основной код
1. Массив и правило элемента заданы
2. Создаётся массив размером по число, которое возвращает функция ElementCounting.
3. Функцией FillNewArray заполняется новый массив.
